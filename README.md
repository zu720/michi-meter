# 🍺 Michi Meter

みちひろのハノイ駐在まで、あと何回飲めるかをカウントするウェブアプリ

## 概要

同僚のみちひろがベトナム・ハノイへの海外駐在が決定！  
出発までに一緒に飲める回数は限られている...そんな思いを形にしたカウントダウンアプリです。

イチロー選手の「イチメーター」にインスパイアされた、友情と別れの記録ツールです。

## 特徴

- 🗼 **ビジュアル進捗**: 東京からハノイまでの道のりを視覚的に表現
- ✈️ **飛行機アニメーション**: 飲むたびにハノイに近づく演出
- 📝 **思い出の記録**: 飲んだ場所、一緒に飲んだ人を記録
- 💾 **自動保存**: ブラウザのlocalStorageで自動的にデータを保存
- 🎯 **25回でゴール**: MAX25回でハノイ到着！

## デモ

[GitHub Pagesでデモを見る](https://yourusername.github.io/michi-meter/)

## 使い方

1. このリポジトリをクローンまたはダウンロード
2. `michi-meter.html` をブラウザで開く
3. 飲み会の後に「場所」と「誰と飲んだか」を記録
4. 25回飲んだらハノイ到着！🎉

## GitHub Pagesでの公開方法

1. GitHubでリポジトリを作成
2. このファイルをプッシュ
3. Settings > Pages > Source を `main` ブランチに設定
4. `michi-meter.html` を `index.html` にリネーム（オプション）

## 技術スタック

- React 18（CDN版）
- Vanilla CSS
- LocalStorage API

## カスタマイズ

`MAX_DRINKS` の値を変更することで、目標回数をカスタマイズできます：

```javascript
const MAX_DRINKS = 25; // この数値を変更
```
